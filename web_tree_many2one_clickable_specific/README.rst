=================================================
Specific clickable many2one fields for tree views
=================================================

Change clickable many2one fields as specific with a wigget: many2one_is_clickable

Configuration
=============
\-

Usage
=====
Add many2one_is_clickable widget to a tree view's field to make
a many2one field clickable.
