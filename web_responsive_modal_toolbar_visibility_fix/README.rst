====================================================
Web Responsive: Modal WYSIWYG Toolbar Visibility Fix
====================================================

* Fix the WYSIWYG editor's toolbar which would get lost behind
  the modal window when editing e.g. Sale Order's outgoing e-mail.
* Forces the Z-index of the toolbar to a high number when a modal
  is visible
