==============================
Change General Tree-view limit
==============================

* Go to Inventory --> Configuration --> Settings and change
  General Tree view limit value to limit the records shown
  on all tree views.

Configuration
=============
* No special configuration needed.

Usage
=====
* Install this module from Apps.

======================
